<b>text</b> algo mostrado em negrito

* LISTAS

<ol reversed = "true"> deixa a ordem da lista invertida, não precisa por o true
<ol type = "I"> Coloque type="I" para algarismos romanos maiúsculos, "i" para algarismos romanos minúsculos, "A" para letras maiúsculas, "a" para letras minúsculas, ou "1" para números, que é o padrão.

<ol start ="6> Se você quiser começar uma lista ordenada, digamos, no 6º item, simplesmente adicione start="6".
